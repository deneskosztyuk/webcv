DSPS - Space Probe Simulator

The "Deep Space Probe Simulator (DSPS)" is a learning tool that simplifies space probe operations for beginners. It uses an ESP32 microcontroller to display real-time telemetry and a Python-based GUI for control. A Wi-Fi system with an HTTP server enables communication, with plans to enhance the UI, create a custom PCB, and add sensors. DSPS promotes hands-on learning in space technology.

Technology used: Python, C, C++

Banking Management System

This Java bank management system uses OOP and data structures to handle user accounts and transactions. The `User` class manages deposits, withdrawals, and stores transaction history in an `ArrayList`, while the `Bank` class uses a `HashMap` to manage accounts. The system supports registration, search, and transactions via a command-line interface, emphasizing efficient data storage and scalable account management.

Technology used: Java, OOP, Data Structures and Algorithms

Artificial Intelligence Custom ML Training

This project developed an AI agent using reinforcement learning with Unity ML-Agents and TensorFlow. The agent was trained to navigate towards a target, improving over time by avoiding obstacles through a reward-penalty system. Despite challenges, it showed significant progress in task efficiency, highlighting reinforcement learning's adaptability in complex environments.

Technology used: C#, Unity, Machine Learning Agents, TensorFlow

Self-Navigating Robot

This project developed a self-navigating robot with three IR sensors and a microcontroller, programmed in C++ for autonomous navigation. Using a left-following algorithm, the robot adjusts its movements based on sensor data to avoid obstacles. The project showcases object-oriented programming and data structures in robotics for efficient maze navigation.

Technology used: C++, OOP, PCB Design & Manufacturing, 3D Design & Manufacturing
